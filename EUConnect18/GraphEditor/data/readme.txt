This folder contains the JSON files read by the graph editor.

graph.JSON - file read by graphEditor.js.  

Accessory files
  Rename one of these file to graph.JSON in order to display in the editor.

graphDefault.JSON - starting point for all students. Minimal nodes and relations.

graph1.JSON  - Possible graph for Attendee #1, with STUDY1-1, PERSON1-1, PERSON1-2, etc.

graph2.JSON -  Possible graph for Attendee #2, with STUDY2-1, PERSON2-1, PERSON2-2, etc.
